# ♾️ Infinity Canvas

A small learning project to explore **canvas drawing**, **smooth curves**, and **2D movement (X/Y axes)** using **vanilla JavaScript**.

---

## 🚀 Features

* ✏️ Smooth freehand drawing
* 🧠 Curve interpolation (natural-looking strokes)
* 🖱️ Mouse-based drawing
* 🧭 Move canvas content using:

  * Right-click + drag
  * Keyboard / button controls (X & Y axis)
* 🎯 SVG path generation from drawn strokes

---

## 📁 Version

**v1** includes:

* Basic drawing system
* Smooth movement across X and Y axes
* Cursor-based panning (right-click)
* Simple UI controls

---

## 🎮 Controls

| Action         | Input              |
| -------------- | ------------------ |
| Draw           | Left click + drag  |
| Move mode      | Right click        |
| Move (X/Y)     | Buttons / keyboard |
| Reset position | Reset button       |
